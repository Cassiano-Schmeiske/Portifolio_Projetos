Identificação de imóveis para compra e revenda a fim de maximizar o lucro

Nesse Projeto foi realizado análise exploratória de dados em Python, afim de identificar imóveis abaixo do preço médio de venda e definição do preço ideal de revenda. Foi desenvolvido e colocado em produção Dashboard interativo com sugestões dos imóveis com maior propensão de Lucro. A análise teve como objetivo aumentar a receita da empresa fictícia, House Rocket, que funciona com a compra e venda de imóveis.

As ferramentas utilizadas foram:

  • Jupyter Notebook, Pycharm
  • Python, Pandas, Numpy, Matplotlib e Seaborn
  • Mapas interativos com Plotly e Folium
  • Heroku Cloud
  • Streamlit Python framework web
